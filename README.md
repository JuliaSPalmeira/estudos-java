## :books: Fontes de Estudo e Base de Conhecimento

A base para os códigos e conceitos aplicados neste repositório veio dos seguintes cursos e plataformas:

*   **DevDojo (YouTube):** Aulas e tutoriais do canal DevDojo para aprofundamento em Java Core e boas práticas de programação.
*   **DIO (Digital Innovation One):** Resolução de listas de exercícios para treinar e consolidar a Lógica de Programação, com base nos desafios propostos por Dienens Chaves.
*   **Pesquisa Ativa:** Exercícios complementares e soluções de problemas encontrados através de pesquisas na internet (Stack Overflow, documentação oficial Java), demonstrando minha capacidade de buscar soluções de forma autônoma.
